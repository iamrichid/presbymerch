# Praisafest Merch Shop Template

A Vue 3 + Vite + Tailwind storefront starter themed around the Praisafest cap artwork.

## Start

```bash
npm install
npm run dev
```

## What is included

- Bold hero section with Praisafest-inspired splash colors
- Featured merch product grid
- Brand story / lookbook section
- Checkout summary block placeholder
- Reusable Tailwind theme tokens for the orange, sky, pink, purple, white, and ink palette
